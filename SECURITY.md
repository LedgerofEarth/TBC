# Security Policy

## Reporting a Vulnerability

**DO NOT** open a public issue for security vulnerabilities.

Instead, please email: security@coreprover.io

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Security Audit

CoreProver undergoes regular security audits.

## Bug Bounty

We offer bug bounties for critical vulnerabilities.

**Rewards:**
- Critical: Up to $50,000
- High: Up to $10,000
- Medium: Up to $2,500
- Low: Up to $500

## Disclosure Policy

We follow a 90-day disclosure policy.

## Security Best Practices

1. Always verify contract addresses
2. Use hardware wallets for large amounts
3. Test on testnet first
4. Monitor for unusual activity
5. Keep private keys secure
